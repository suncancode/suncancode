Three ways to add styling to HTML Elements

CSS can be applied in 3 ways:

  1. Inline CSS: Applied directly within HTML elements using the sytle attribute.
  2. Document (Internal) CSS: Defined within a <style> tag in the HTML <head> section.
  3. External CSS: Stored in a separate ".css" file and linked to the HTML file using a <link> tag.

Key CSS Concepts

Selectors

  • Element Selector: Target HTML elements (e.g., { color: blue; }).
  • Class Selector: Target elements with a specific class (e.g., .userinfo { color: purple; }).
  • ID Selector: Targets an unique element by ID (e.g., #userHeading { color: orange; }).

Precedence Rules

CSS follows a hierarchy:

  • Inline CSS overrides Document CSS.
  • Document CSS overrides External CSS.
For example, if an external CSS sets p { color: black; }, a document CSS can override it with p { color: blue; }, and inline CSS can further override with this text.

Common Properties

  • color: Sets text color (e.g., color: #D3D3D3; for light grey).
  • font-family: Sets font (e.g., font-family: Courier;).
  • font-size: Sets text size (e.g., font-size: 150%;).
  • border: Adds a border (e.g., border: 1px solid black;).
  • padding: Adds space inside an element (e.g., padding: 10px;).
  • background-color: Sets background color (e.g., background-color: lightgrey;).
