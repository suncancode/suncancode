6. Table

Attributes:

border: Sets border thickness (e.g., border="1")

width: Sets column width (e.g., width="50%" or width="150px")

align: Aligns content horizontally (left, center, right)

valign: Aligns content vertically (top, middle, bottom)

<tr>: Table row

<th>: Table header

<td>: Table data

<caption>: Table caption

Example:

User Information
Username First Name Last Name
jsmith John Smith
mlee Mary Lee

Annotation: The caption provides a title for the table, and attributes like align enhance presentation.

7. Formatting Text

Example:

Bold: In đậm (bold).

Italic: In nghiêng (italic).

Highlighted: Đánh dấu (highlight).

Strikethrough: Gạch ngang (deleted text).

Underlined: Gạch chân (inserted text).

Annotation: These tags add semantic meaning and visual styling to text.

Attributes

1. Popular attributes

id: Unique identifier for an element, used for linking or applying CSS

href: Link destination

src: Path to a resource (e.g., image, file)

alt: Alternative text for images

target: Specifies how to open a link (e.g., target="_blank")

Annotation: Attributes enhance tag functionality and accessibility.

2. Character entity

Some characters are reserved in HTML. Reserved characters in HTML must be replaced with character entities.

Character Entity Meaning
(non-breaking space) &nbsp; Non-breaking space
< &lt; Is less than
> &gt; Is greater than
& &amp; Ampersand
" &quot; Double quote
' &apos; Single quote (apostrophe)
° &deg; Degree
© &copy; Copyright

Annotation: Entities like &nbsp; prevent line breaks, while others represent reserved characters.
