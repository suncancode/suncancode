Lecture 8 - JSON

1. Introduction to JavaScript Arrays

A JavaScript Array is a special variable that can hold multiple values in a single structure. Arrays are useful for storing lists of items, such as names, numbers, or objects. They are dynamic, meaning you can add or remove elements after creation.

2. Creating and Initializing Arrays

Arrays can be created using square brackets []. You can initialize an array with values or leave it empty. 

var arrayName = [item0, item1, ...];
var emptyArray = [];
var subjectList = ["ISIT206", "MATH121", "CSCI301"];

- arrayName: A variable holding the array. - emptyArray: An empty array. - subjectList: An array with initial values.

3. Modifying Array Elements

You can modify array elements by assigning new values using their index (starting from 0). Adding elements beyond the current length creates "holes" (undefined spaces) in the array. 

subjectList[1] = "LOGIC101"; // Changes the second element
subjectList[6] = "LAW201";   // Creates holes in the array

- After subjectList[1] = "LOGIC101", the array becomes ["ISIT206", "LOGIC101", "CSCI301"]. - subjectList[6] = "LAW201" adds a value at index 6, leaving indices 3, 4, and 5 as undefined.

4. Looping Through an Array

You can iterate over array elements using a for loop with the length property to access each item. 

for(var i = 0; i < subjectList.length; i++) {
    alert(subjectList[i]);
}

This loop displays each element of subjectList in an alert box. For the array ["ISIT206", "LOGIC101", "CSCI301", undefined, undefined, undefined, "LAW201"], it will show each value, including undefined for holes.

5. JSON Stringify and Parse

JSON (JavaScript Object Notation) is a lightweight data interchange format. Two key methods allow conversion between JavaScript objects/arrays and JSON strings:

  • JSON.stringify(): Converts a JavaScript object or array into a JSON string.
  • JSON.parse(): Parses a JSON string back into a JavaScript object or array.
var subjectList = ["ISIT206", "LOGIC101", "CSCI301"];
var jsonString = JSON.stringify(subjectList);
console.log(jsonString); // Output: ["ISIT206","LOGIC101","CSCI301"]

var parsedArray = JSON.parse(jsonString);
console.log(parsedArray); // Output: ["ISIT206","LOGIC101","CSCI301"]

- JSON.stringify() is useful for sending data to a server. - JSON.parse() is used to process JSON data received from a server or stored locally.
