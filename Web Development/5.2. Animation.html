1. var vs let

var: Function-scoped, hoisting. let: Block-scoped, không hoisting.


function varVsLetExample() {
    if (true) {
        var varVar = "I'm function-scoped";
        let letVar = "I'm block-scoped";
    }
    console.log(varVar); // OK
    // console.log(letVar); // Lỗi
}

Mở console để xem (F12).

2. Template Literals

Chuỗi động với backticks và ${}.


var subject = "CSIT884";
var message = `This is ${subject} subject.`;
console.log(message); // "This is CSIT884 subject."

Mở console để xem.
