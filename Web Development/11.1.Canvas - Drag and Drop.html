Lecture 11A - Graphic Canvas and Drag & Drop

1. Introduction to HTML5 Canvas

The <canvas> element, first introduced by Apple in WebKit for OS X Dashboard, allows dynamic rendering of 2D graphics using JavaScript. It is now supported by major browsers (Chrome, Firefox, Safari, Edge).

  • Used to draw shapes, text, and images on the fly.
  • Requires JavaScript for rendering (e.g., getContext("2d")).
  • Key properties: width and height.

2. Basic Canvas Operations

  • fillRect(x, y, width, height): Draws a filled rectangle.
  • strokeRect(x, y, width, height): Draws a rectangle outline.
  • beginPath(): Starts a new path for drawing.
  • arc(x, y, radius, startAngle, endAngle): Draws a circle or arc.
  • fillText(text, x, y): Adds text to the canvas.
  • drawImage(image, x, y): Draws an image on the canvas.

3. Introduction to Drag and Drop

HTML5 Drag and Drop API enables draggable elements and drop zones using JavaScript events.

  • draggable="true": Makes an element draggable.
  • Key events: dragstart, dragover, drop.
  • Common use: Moving images or elements across the page.

4. Drag and Drop Events

  • ondragstart: Triggered when dragging begins.
  • ondragover: Triggered when an element is dragged over a drop zone (prevent default behavior).
  • ondrop: Triggered when an element is dropped.

5. Practical Applications

Combine Canvas and Drag & Drop to create interactive web applications, such as:

  • Dynamic drawing tools.
  • Image manipulation and repositioning.
  • Game development (e.g., moving characters).
