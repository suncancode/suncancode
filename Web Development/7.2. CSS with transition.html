1. Introduction and Objectives

This lecture focuses on integrating JavaScript with CSS to create interactive and animated effects on web pages. The key objectives are:

  • Use JavaScript to manipulate CSS properties dynamically.
  • Produce animated interactive effects using CSS properties.

2. Manipulating CSS with JavaScript

JavaScript allows you to change CSS properties of HTML elements in real-time. For example, you can modify the background color or size of an element based on user actions. 

document.getElementById("myElement").style.backgroundColor = "blue";

This code changes the background color of an element with `id="myElement"` to blue when executed.

3. Creating Animated Effects with CSS Transitions

CSS Transitions provide smooth animations when CSS properties change. You can define the duration and timing function (e.g., `ease`, `linear`). 

.element {
    width: 100px;
    transition: width 1s ease;
}
.element:hover {
    width: 200px;
}

When hovering over the element, its width increases smoothly from 100px to 200px over 1 second.

4. Using CSS Transforms

CSS Transforms allow you to rotate, scale, or translate elements. Combined with Transitions, you can create dynamic effects. 

.element {
    transition: transform 0.5s;
}
.element:hover {
    transform: rotate(90deg) scale(1.2);
}
